# TT-RSS ToggleToggle Plugin

Add hotkeys to toggle "Show articles" and "Sort articles" settings.

## Hotkeys
  * `` ` `` => Toggle between Adaptive and Starred view
  * `z` => Toggle between Newest first and Oldest first
